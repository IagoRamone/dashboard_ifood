Projeto dashboard_ifood.

--- Front-end ---
    Criado em Next.js utilizando typescript e tailwind.

Estrutura do projeto:

    Página inicial de apresentação do serviço (Não Prioritária)
    Página de login para confirmação de credencial(Importante)
    Página de HomePage criada para dar um panôrama geral sobre a marca(Importante)

--- Back-end ---
    Criado em Python utilizando flask.

Bibliotecas utilizadas:

    Time
    Request
    Dotenv

EStrutura do projeto:

     Conexão com o ifood com dois endpoints Settlements e conciliation. ALém da requisição da requisição para o token que tem o tempo de vida de 3 horas antes de expirar. 
     Atualmente o token é salvo em uma variável que é apagado após o fim da sessão.
